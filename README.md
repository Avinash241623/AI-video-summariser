# 🎥 AI Video Summarizer

An AI-powered web app that summarizes YouTube videos using NLP and OpenAI.

## 🚀 Features
- Extracts YouTube transcripts
- Generates AI summary
- Clean Streamlit UI
- Fast and easy to use

## 🛠 Tech Stack
- Python
- Streamlit
- OpenAI API
- youtube-transcript-api

## 📦 Installation

```bash
git clone https://github.com/Avinash241623/AI-video-summariser.git
cd AI-video-summariser
pip install -r requirements.txt
